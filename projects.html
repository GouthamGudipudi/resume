Project 1: BATTERY CHARGER USING SCR

The battery is charged with a small amount of AC voltage or DC voltage, so; if you want to charge your battery with AC source then should follow these steps, we need first to limit the large AC voltage, need to filter the AC voltage to remove the noise, regulate and get the constant voltage and then give the resulting voltage to the battery for charging. Once charging is completed the circuit should automatically turn off.

Application: It is a silicon-controlled rectifier-based battery charger used for charging 12v batteries, Application for bike battery.

Project 2: HEALTH MONITORING SYSTEM

In this project, we displayed the pulse rate values and temperature values on serial monitor as well as in the Bluetooth terminal in the android phone. A lot of patients die due to heart attacks and heart diseases every year. This project is helpful for sensing and recording heart rate in bpm and temperature in Fahrenheit.

Application: It is based on Bluetooth technology used for the measurement of temperature, Heart rate of a person, Application for Hospitals.

Project 3: IOT BASED WEATHER MONITORING SYSTEM USING NODE MCU

This project involves creating a live weather station using NodeMCU ESP8266. It connects sensors like DHT11, BMP180, and FC37 to measure humidity, temperature, pressure, and rainfall. Data is uploaded to a web server, accessible via NodeMCU's IP address, enabling remote monitoring in various environments like remote areas or freezers.

Application: It is an IOT based project for monitoring weather conditions in environment, it is best application for farmers.

Project 4: GSM BASED MOTOR CONTROL

This project aims to alleviate the challenges faced by Indian farmers, particularly the inconvenience of traveling long distances daily to irrigate their crops. By implementing GSM-based agriculture motor control, farmers can remotely operate irrigation systems via a smartphone app. This technology saves time, money, and effort, providing a more efficient solution for irrigation management.

Application: It is an GSM based project for proper utilization of water for the crops, GSM send message to farmer to control the field water pump from the home, it is also the best application for farmers.
